Tentang Saya

Halo! Saya Dede Abdur Rafi Fauzan, mahasiswa Teknik Informatika semester akhir dengan pengalaman magang sebagai Web Developer di instansi pemerintahan serta pengalaman pengembangan berbagai aplikasi web dan mobile dalam proyek akademik maupun mandiri.

Saya berfokus pada pengembangan aplikasi Full Stack menggunakan Laravel, Next.js, dan Flutter, serta memiliki pemahaman dasar pada cloud computing, networking, dan cybersecurity. Saya juga memiliki pengalaman dalam penelitian pengembangan ulang Content Management System (CMS) untuk optimalisasi layanan akademik yang dipublikasikan pada Seminar APDOVI 2025.

Saya tersertifikasi dalam bidang networking dan security melalui Cisco Networking Academy (CCNA Series & Ethical Hacker), serta memiliki pengalaman hands-on cloud melalui Google Cloud Computing Foundations dan Google Cloud Skill Badges.
